# PinguMall - 电商网站

一个功能完整的电商网站，仿照 pingubuy.com 设计，包含现代化的用户界面和完整的购物功能。

## 项目特点

- 📱 **响应式设计** - 适配各种屏幕尺寸
- 🛒 **完整购物流程** - 从浏览商品到购物车结算
- 👤 **用户认证系统** - 注册、登录、个人中心
- 🔍 **智能搜索** - 支持商品搜索和筛选
- 🎨 **现代化UI** - 使用 Bootstrap 5 和自定义样式
- ⚡ **交互体验** - 丰富的动画效果和用户反馈

## 项目结构

```
PinguMall/
├── index.html          # 网站主页
├── login.html          # 登录页面
├── register.html       # 注册页面
├── products.html       # 商品列表页
├── cart.html           # 购物车页面
├── css/

├── js/
│   ├── main.js         # 主页功能
│   ├── auth.js         # 认证功能
│   ├── products.js     # 商品页功能
│   └── cart.js         # 购物车功能
└── README.md           # 项目说明
```

## 主要功能

### 🏠 主页 (index.html)
- 轮播图展示
- 商品分类导航
- 热门商品推荐
- 搜索功能

### 🔐 用户认证
- **注册页面** - 完整的用户注册表单
- **登录页面** - 用户登录验证
- **表单验证** - 实时验证用户输入
- **数据持久化** - 使用 localStorage 存储

### 🛍️ 商品页面 (products.html)
- 商品列表展示
- 多维度筛选（分类、价格、品牌）
- 排序功能（价格、名称、评分）
- 分页显示
- 网格/列表视图切换
- 商品详情模态框

### 🛒 购物车 (cart.html)
- 商品管理（增删改查）
- 全选/单选功能
- 数量调整
- 价格计算
- 优惠券系统
- 配送方式选择
- 推荐商品

## 技术栈

- **HTML5** - 语义化标签
- **CSS3** - 自定义样式 + Bootstrap 5
- **JavaScript (ES6+)** - 原生 JS 实现所有功能
- **Bootstrap 5.1.3** - UI 框架
- **Font Awesome 6.0** - 图标库
- **localStorage** - 本地数据存储

## 开始使用

1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html`
3. 开始体验购物功能

## 功能演示

### 注册新用户
1. 点击导航栏的"注册"
2. 填写注册表单
3. 系统会验证表单并创建用户

### 浏览和购买商品
1. 在主页浏览推荐商品
2. 使用搜索框搜索特定商品
3. 在商品页面使用筛选器
4. 点击"加入购物车"添加商品

### 管理购物车
1. 点击导航栏的购物车图标
2. 调整商品数量或删除商品
3. 选择配送方式
4. 使用优惠券（测试代码：SAVE10、FIRST20）
5. 进入结算流程

## 测试数据

### 优惠券代码
- `SAVE10` - 新用户9折优惠
- `FIRST20` - 首单8折优惠
- `FREE15` - 免配送费

### 商品分类
- 电子产品
- 时尚服饰
- 家居生活
- 运动户外

## 开发说明

项目使用原生 JavaScript 开发，所有数据存储在浏览器的 localStorage 中。在实际部署时，需要：

1. 连接真实的后端API
2. 实现用户认证和授权
3. 添加支付系统集成
4. 优化 SEO 和性能

## 自定义和扩展

### 添加新商品
编辑 `js/main.js` 中的 `featuredProducts` 数组

### 修改样式
编辑 `css/style.css` 文件自定义样式

### 添加新功能
在对应的 JavaScript 文件中添加新函数
